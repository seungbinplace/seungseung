.red라는 클래스를 사용하면 바탕화면이 붉으낵이 됩니다.

```html
<p class="red">배경이 붉은색이 됩니다.</p>
```

css 내용은 아래와 같습니다.
```css
.red{
    background-color:red;
}
```
